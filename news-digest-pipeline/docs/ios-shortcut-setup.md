# iOS Shortcut Setup — News Digest

## Назначение
Shortcut появляется в Share Sheet на iPhone. При нажатии "Поделиться" в Perplexity → выбираешь "News Digest" → URL статьи автоматически отправляется на сервер.

## Endpoint
```
POST https://news.questtales.com/api/articles
Content-Type: application/json
Body: {"url": "https://..."}
```

## Пошаговая инструкция (iOS 26)

### Шаг 1. Создать новую команду
1. Открой приложение **Shortcuts** (Команды) на iPhone
2. Нажми **"+"** вверху экрана для создания новой быстрой команды
3. Нажми **"Добавить действие"**

### Шаг 2. Добавить блок "Получить URL"
1. В поиске набери **"URL"**
2. Из списка выбери **"Получить URL адреса из входных данных"**
3. Этот блок появится на экране. Слово "входные данные" внизу — не трогай, оставь как есть

### Шаг 3. Добавить блок "Получить содержимое URL"
1. Нажми **"+"** внизу экрана для добавления второго действия
2. В поиске набери **"Содержимое"**
3. Выбери **"Получить содержимое URL"**
4. Блок появится на экране. Внутри него будет поле с надписью **"URL адреса"**

### Шаг 4. Настроить адрес сервера
1. Нажми на надпись **"URL адреса"** внутри второго блока
2. Откроется поле ввода. **Важно:** НЕ нажимай на "Выбрать переменную" или "Имя переменной" — если такой экран открылся, нажми "Отменить" и попробуй снова
3. Сотри содержимое поля и напечатай: `https://news.questtales.com/api/articles`
4. Система спросит "Разрешить этой команде подключиться к news.questtales.com?" — нажми **"Разрешить"**

### Шаг 5. Настроить HTTP-запрос
1. Под блоком "Получить содержимое URL" найди маленькую **стрелочку вниз (˅)** или слово **"Ещё"** — нажми на неё
2. Откроются дополнительные настройки:
   - **Метод:** нажми на "GET", смени на **"POST"**
   - **Заголовки:** нажми чтобы раскрыть, затем нажми **плюсик** "Добавить новый заголовок"
     - Левое поле (ключ): `Content-Type`
     - Правое поле (значение): `application/json`
   - **Тело запроса:** уже стоит "JSON" — оставь как есть. Нажми **плюсик** "Добавить новое поле"
     - Выбери тип **"Текст"**
     - Левое поле (ключ): `url`
     - Правое поле (значение): **не печатай руками!** Нажми на поле, над клавиатурой появится панель с переменными. Найди **"URL адреса"** (результат из первого блока) — нажми на неё. Вставится голубая капсула

### Шаг 6. Добавить уведомление (необязательно, но полезно)
1. Нажми **"+"** для добавления третьего действия
2. В поиске набери **"Уведомление"**
3. Выбери **"Показать уведомление"**
4. В текстовом поле замени стандартный текст на: `Статья отправлена`

### Шаг 7. Переименовать команду
1. Нажми на название команды вверху экрана (один раз, не удерживай)
2. Появится меню — выбери **"Переименовать"**
3. Набери: `News Digest`

### Шаг 8. Включить Share Sheet (Лист отправки)
**Внимание:** в iOS 26 эта настройка называется по-другому, чем в документации!

1. Из редактора команды (экран с блоками) перейди в настройки shortcut
2. Найди пункт **"На странице экспорта"** (это и есть Share Sheet, просто другое название в iOS 26)
3. **Включи** этот переключатель
4. Нажми **"Готово"**

**Как найти настройки:** нажми и удерживай на команде "News Digest" в списке команд → выбери **"Подробнее"** → прокрути до пункта "На странице экспорта"

## Проверка
1. Открой любую статью в **Perplexity** на iPhone
2. Нажми **"Поделиться"** (кнопка Share)
3. В списке найди **"News Digest"** — нажми
4. Если всё настроено правильно — появится уведомление **"Статья отправлена"**
5. Проверить что сервер получил статью:
   ```
   curl https://news.questtales.com/api/articles/stats
   ```

## Troubleshooting

| Проблема | Решение |
|----------|---------|
| Shortcut не появляется в Share Sheet | Проверь что включён переключатель "На странице экспорта" в настройках shortcut |
| Ошибка "URL не указан" | Проверь что в блоке "Получить содержимое URL" стоит адрес `https://news.questtales.com/api/articles`, а НЕ переменная "URL адреса" |
| Ошибка сети | Проверь что URL начинается с https (не http) |
| Сервер не отвечает | Открой в браузере https://news.questtales.com/health — должен показать JSON со статусом |
| "Разрешить подключение?" не появлялось | Удали shortcut, создай заново — запрос появится при первой настройке адреса |

## Итоговая структура команды

После настройки в редакторе должно быть **3 блока** (сверху вниз):
1. **Получить URL адреса из входных данных** — без настроек
2. **Получить содержимое URL** — адрес сервера, POST, JSON с ключом url
3. **Показать уведомление** — "Статья отправлена"
