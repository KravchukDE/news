# Instagram Pipeline — Рабочий документ

## Цель

Автоматическая публикация новостных дайджестов в Instagram (@alexeykrol) с уникальными изображениями и кликбейтными заголовками.

## Ограничения Instagram

- **Caption**: максимум 2200 символов (включая хэштеги)
- **Изображение**: квадрат 1080x1080 или 4:5 (1080x1350) — лучший формат для ленты
- **Хэштеги**: до 30 штук, рекомендуется 5-15
- **Комментарии**: можно добавлять программно через API (для остатка текста)
- **API**: Instagram Graph API через Facebook App (Business/Creator аккаунт)
- **Текст на изображении**: Instagram не индексирует текст на картинках, но он привлекает внимание в ленте

## Архитектура пайплайна

```
Дайджест (готовый текст)
    │
    ├── 1. Генерация промпта для картинки
    │       Claude API: из дайджеста → короткий кликбейтный заголовок + промпт для изображения
    │
    ├── 2. Выбор шаблона-референса (рандомно из 2-10 шаблонов)
    │
    ├── 3. AI генерация изображения (img2img на базе шаблона + промпт)
    │       Сервис: [TBD — результат исследования]
    │
    ├── 4. Наложение текста на изображение
    │       - Кликбейтный заголовок (крупный шрифт, кириллица)
    │       - Программно (Sharp/Canvas), идеальная орфография
    │       - Формат: 1080x1350 (4:5)
    │
    ├── 5. Подготовка caption
    │       - Первые 2000 символов дайджеста (или сокращённая версия)
    │       - Хэштеги в конце
    │       - Остаток текста → комментарии
    │
    └── 6. Публикация
            - Instagram Graph API или Patchright
            - Первый комментарий с продолжением текста (если нужно)
```

## Этапы реализации

### Этап 1: Генерация изображений (текущий)

**Задача:** научиться создавать уникальные изображения на базе шаблонов-референсов.

**Шаги:**
1. Создать 2 тестовых шаблона (стиль, палитра, композиция)
2. Выбрать сервис генерации (результат исследования)
3. Написать скрипт: шаблон + промпт → уникальное изображение
4. Протестировать на 5-10 вариациях

**Критерии готовности:**
- Изображение визуально качественное (не «AI-мусор»)
- Стиль консистентный (узнаваемый бренд)
- Генерация занимает < 30 секунд
- Работает через API без ручного вмешательства

### Этап 2: Наложение текста

**Задача:** программно накладывать кликбейтный заголовок на сгенерированное изображение.

**Шаги:**
1. Выбрать шрифт (жирный, читаемый, поддержка кириллицы)
2. Написать скрипт наложения (Sharp или node-canvas)
3. Обработать edge cases: длинный текст, перенос строк, контраст с фоном
4. Добавить подложку/тень для читаемости текста на любом фоне

**Критерии готовности:**
- Текст 100% читаемый на любом фоне
- Орфография идеальная (текст приходит от Claude, не от генератора изображений)
- Формат 1080x1350 (4:5)

### Этап 3: Публикация в Instagram

**Задача:** автоматическая публикация картинка + caption.

**Два подхода:**
- **Instagram Graph API** — нужен Business/Creator аккаунт + Facebook Page привязка
- **Patchright** — тот же подход что с Facebook Profile

**Шаги:**
1. Определить подход (API или Patchright)
2. Caption: первые ~2000 символов + хэштеги
3. Тест публикации
4. Если текст > 2000: остаток в первый комментарий

### Этап 4: Интеграция в основной пайплайн

**Задача:** Instagram публикуется автоматически вместе с Telegram и Facebook.

## Промпт для генерации заголовка

```
На основе этого дайджеста создай:
1. Кликбейтный заголовок (5-8 слов, русский язык) — для наложения на изображение
2. Промпт для генерации фонового изображения (на английском, 1 предложение)

Заголовок должен:
- Вызывать любопытство
- Быть провокационным, но не ложным
- Содержать конкретику (цифры, имена)

Пример: "ИИ уволил 80% отдела. Босс не жалеет."
Пример: "Stack Overflow мёртв. Что дальше?"
```
