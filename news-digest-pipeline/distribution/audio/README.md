# Audio Pipeline

**Статус:** Планируется

## Концепция

Аудиоверсия дайджеста — подкаст-формат. Полный текст озвучивается, добавляется intro/outro, публикуется на подкаст-платформы и в Telegram.

## Пайплайн

1. Дайджест → адаптация текста для аудио (убрать ссылки, хэштеги)
2. TTS озвучка полного текста
3. ffmpeg: intro + озвучка + outro + нормализация громкости
4. Публикация: Telegram voice, подкаст-платформы
