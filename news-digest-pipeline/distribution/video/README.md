# Video Pipeline

**Статус:** Планируется

## Концепция

Автоматическая генерация коротких видео (60-90 сек) из дайджеста для YouTube Shorts, Instagram Reels, TikTok.

## Пайплайн

1. Дайджест → Claude → сценарий (ключевые тезисы, 5-8 слайдов)
2. TTS озвучка (ElevenLabs / Google TTS / локальный)
3. Генерация визуального ряда (слайды с тезисами + фоновые изображения)
4. ffmpeg сборка: аудио + визуал + переходы + субтитры
5. Публикация на платформы
